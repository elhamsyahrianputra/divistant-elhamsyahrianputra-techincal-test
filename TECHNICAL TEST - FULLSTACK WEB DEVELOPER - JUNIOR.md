# **Context**

Buat aplikasi pengelolaan sederhana koleksi buku menggunakan frontend JS dan backend Node.js-based.  
Ada halaman untuk melihat semua buku, dan bisa login untuk mendapatkan fungsi CRUD.

# **Frontend Specifications**

1. Base: **Vue / React** (Vue preferred, bebas menggunakan framework apapun on top).  
2. Halaman & fitur  
- Halaman listing untuk semua buku  
- Halaman detail lengkap buku  
- Halaman login & fungsi logout  
- Fungsi tambahan CRUD untuk yang sudah login:  
* Tombol & halaman untuk buat buku  
* Tombol & halaman edit buku  
* Tombol delete buku  
3. State management bebas jika ingin menggunakan library apapun.  
4. Styling tampilan bebas menggunakan style framework/theme apapun.

# **Backend Specifications**

1. Base: **Node.js** (bebas menggunakan framework apapun on top).  
2. Authentication:  
- JWT-based.  
- Lindungi beberapa endpoint non-public menggunakan middleware.  
3. API endpoints:  
- Standard CRUD untuk buku.  
4. Data storage:  
- Gunakan in-memory database (SQLite, Redis, dll) atau SQL database (Postgres, MySQL).  
- Sertakan dummy/demo database yang sudah berisi beberapa data supaya setelah setup aplikasi bisa langsung dilihat datanya.
